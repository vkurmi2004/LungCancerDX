🧠 Ensemble Architecture

Model Suite

Five state-of-the-art convolutional neural networks fine-tuned on lung CT data, combined via weighted soft-voting.

⚖️
Soft-Voting Ensemble

Each model's class probability vector is multiplied by its learned weight. The weighted average determines the final prediction, improving robustness over any single model.
